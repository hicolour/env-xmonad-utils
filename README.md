env-xmonad-utils
================

Xmonad utils
    
* xspotify - Spotify controller using D-Bus.
    (D-Bus should be available by default as it is a dependency of systemd).
      
        * play | next | prev | status* | now*
        
        * status & now are extra functions which can be used for dzen2
          eg.
              now now
              echo $now
